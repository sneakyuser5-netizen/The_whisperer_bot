# 🤖 WhisperBot

A fast, lightweight and modular WhatsApp bot built with **Node.js** and **Baileys**.

WhisperBot focuses on group management, moderation, utility commands and multilingual support while remaining easy to extend with plugins.

---

## ✨ Features

- 🌍 English & French support
- 👮 Group administration
- 🔗 Anti-link protection
- 👋 Welcome & goodbye messages
- 🏷️ Tag all members/admins
- ⚡ Fast command handler
- 🔐 Public / Private mode
- 👑 Owner & Sudo system
- 📊 Runtime & server information
- 🧩 Plugin-based architecture
- 💬 Fun commands
- 🛠️ Utility tools

---

## 📂 Command Categories

- Admin

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/sneakyuser5-netizen/The_whisperer_bot.git
```

Enter the project

```bash
cd The_whisperer_bot
```

Install dependencies

```bash
npm install
```

Start the bot

```bash
npm start
```

---

## 🔧 Configuration

Configure your bot settings inside:

```
config.js
```

## 🌐 Language Support

WhisperBot currently supports:

- 🇬🇧 English
- 🇫🇷 French

Translations are automatically generated using:

```bash
npm run translate
```
